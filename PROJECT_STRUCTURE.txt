API ENDPOINTS (25+)
================================================================================

Product-AI (Port 3001):
  POST /generate-seo               Generate SEO metadata
  POST /generate-seo/bulk          Bulk SEO generation
  POST /validate-seo               Validate SEO
  POST /generate-description       Generate descriptions
  POST /enhance-description        Enhance existing
  POST /generate-features          Extract features
  POST /suggest-tags               Suggest tags
  POST /merge-tags                 Merge tag lists
  POST /categorize-tags            Categorize tags
  POST /validate-tags              Validate tags
  POST /bulk-optimize              Bulk optimization

Inventory-AI (Port 3002):
  POST /forecast-demand            Demand forecasting
  POST /calculate-reorder          Reorder point
  POST /calculate-safety-stock     Safety stock
  POST /calculate-eoq              Economic Order Quantity
  POST /detect-seasonality         Seasonality detection
  POST /stockout-probability       Stockout risk

Workflow Engine (Port 3003):
  POST /create-approval            Create approval
  POST /approvals/:id/approve      Approve item
  POST /approvals/:id/reject       Reject item
  GET  /approvals/pending/:shopId  Get pending
